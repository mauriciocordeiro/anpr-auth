CatEye ALPR v1.0

Usage:

  1. Send a POST http request to "/alpr" with a image file (input "file", enctype "multipart/form-data") to detect a plate
  2. Use the link "/upload" to select a file to send and detect
  3. Use ours ImageZQM service (port 5556) to send images do detect (Take a look at the file "client.py to see an example")

Response:

A JSON string like:

{"type": "string", "number": "string:\w(3)\d[\w\d]\d(2)", "confidence": float, "time": float, "roi": array:[x, y, width, height]}
